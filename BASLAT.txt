→ Python yüklü mü? Kontrol et: python --version
→ PATH'e eklemeyi unutmuş olabilirsin

Hata 2: "Flask bulunamadı"
→ Terminal'de yaz: pip install Flask

Hata 3: "Port kullanımda"
→ app.py dosyasında port=5001 yap

Hata 4: Sayfa açılıyor ama stilsiz
→ Tarayıcıyı yenile: Ctrl+F5
→ Önbelleği temizle: Ctrl+Shift+Del
